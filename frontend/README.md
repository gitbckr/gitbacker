# Frontend

Next.js web application (App Router) with shadcn/ui, Tailwind CSS, and TanStack Query.

## Prerequisites

- Node.js 18+
- pnpm
- Backend API running on port 8000

## Setup

```bash
pnpm install
```

## Run

```bash
pnpm dev
```

Opens on [http://localhost:3000](http://localhost:3000).

## Build

```bash
pnpm build
pnpm start
```
